Setup Instructions for RPG Playmaker Scene:

Before you can view the RPG Playmaker Scene, you must first turn on
Playmaker support, then import the Playmaker Global Variables used by
the RPG Scene. Follow the steps below.

1) Turn on Playmaker Support.  
   * Go to "Build Settings -> Player Settings" for the platform you
     are using and add "GDE_PLAYMAKER_SUPPORT" to the Scripting Define
     Symbols.

2) Import Playmaker Global Variables.  
   * In the Unity Project view, browse to "GameDataEditor -> Playmaker
     -> Scenes -> RPGExample" folder.
   * Select the PlaymakerGlobals_EXPORTED.asset file.  
   * In the Inspector view, click "Import Globals"
